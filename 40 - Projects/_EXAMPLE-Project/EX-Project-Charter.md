# Example Project — Charter

> A generic, topic-neutral example of a **project charter** note. Copy this folder (`_EXAMPLE-Project`) to start a real project.

## Goal
State the single outcome this project will deliver, in one sentence.

## Scope
- **In scope:** what this project will do.
- **Out of scope:** what it deliberately will not do.

## Definition of done
A short checklist that, when complete, means the project can move to `70 - Archive`.

- [ ] Outcome delivered
- [ ] Reviewed
- [ ] Archived
