# 📚 MOC — Literature

**Literature notes** capture, in your own words, what a single source says. They are the bridge between reading and your [[Zettelkasten]] permanent notes. See [[SQ3R]] and [[Progressive Summarization]].

## Worked example
A generic relational pair showing how a literature note feeds a permanent note:

- [[EX-Literature-Note]] — your paraphrase of an example source.
- [[EX-Permanent-Note-A]] — the atomic idea distilled from it.

The literature note links forward to the permanent note it produced.

## How to add a source
Use [[Template-Literature-Note-Paper]] (papers) or [[Template-Book-Note]] (books).
