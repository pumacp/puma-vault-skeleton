# 🔌 Recommended Obsidian Plugins (all optional)

> [!info]
> The skeleton ships **no plugin configuration** and works with none installed. These are *optional* community plugins that pair well with the vault. Install only what helps you. See [[Getting-Started]].

## Useful first
| Plugin | Why |
|--------|-----|
| **Dataview** | Query the vault like a database (dynamic tables of tasks, notes). |
| **Templater** | Fill the note-type [[MOC Templates|templates]] automatically; set the template folder to `00 - Meta/Templates`. |
| **Tasks** | Track checkboxes across the vault with due dates and filters (powers [[TASKS-Active]]). |
| **Calendar + Periodic Notes** | Daily/weekly notes from a calendar; point them at `90 - GTD/95 Reviews`. |
| **Obsidian Git** | Version and back up the vault to GitHub (see [[Git-GitHub-Setup]]). |
| **QuickAdd** | One-shortcut note creation from a template. |

## Nice to have
- **Spaced Repetition** — review permanent notes over time (see [[Spaced Repetition]]).
- **Excalidraw** — sketch diagrams and concept maps.
- **Smart Connections** — AI-suggested related notes (needs an API key or a local model).
- **Advanced Tables**, **Tag Wrangler**, **Auto Link Title**, **Front Matter Title** — quality-of-life.

## Suggested core settings
```
Default new note location: 10 - Inbox
Use [[Wikilinks]]: ON · Always update internal links: ON
Attachments folder: 00 - Meta/Attachments
```
