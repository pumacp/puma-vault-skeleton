# Keshav Three-Pass: {{title}}

> **Keshav's Three-Pass Method** — read a paper in up to three passes. Each pass maps to a [[Zettelkasten]] note type.

## Pass 1 — Bird's eye view (5–10 min) → fleeting note
Read title, abstract, headings, conclusions. Answer the **Five Cs**:

| C | Answer |
|---|--------|
| Category | type of paper |
| Context | related work / theory used |
| Correctness | do the assumptions hold? |
| Contributions | main contributions |
| Clarity | is it well written? |

**Decision:** relevant → Pass 2 · not relevant → archive.

## Pass 2 — Grasp the content (≤ 1h) → literature note
Read carefully, skip proofs. Capture key points, figures, and references to follow up. This becomes your literature note.

## Pass 3 — Virtual re-implementation (4–5h) → permanent note
For core papers only: reconstruct the argument from scratch, surface hidden assumptions, then write a [[Zettelkasten|permanent note]] of the synthesized insight.

## Fleeting → Literature → Permanent
- Fleeting: initial capture in `10 - Inbox`
- Literature: this note (Pass 1–3)
- Permanent: synthesized insight in `30 - Permanent`
