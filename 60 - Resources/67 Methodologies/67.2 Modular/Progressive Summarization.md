# Progressive Summarization

> Distill a source in layers — keep the highlights of your highlights — so a note stays skimmable yet preserves the full original when you need it.

## How it works
Each pass compresses the previous one, only when you revisit the note:

1. Save the source as a literature note.
2. **Bold** the passages that matter.
3. **Highlight** the best of the bold.
4. Write a one-line executive summary in your own words.

You never delete the lower layers, so detail is recoverable; you mostly read the top layer.

## Source
Tiago Forte, "Progressive Summarization" (Forte Labs, 2017).

## Bridge
This *is* the **Distill** phase of [[CeReBro (CODE)]] made concrete, and it is the mechanism that turns a literature note into a [[Zettelkasten]] permanent note. Opt in whenever you process long sources.

## Worked examples
- **Example 1 — A dense paper.** A 12-page paper's literature note is bolded on second read, top-highlighted on third, and capped with a one-sentence claim that becomes its own permanent note.
- **Example 2 — Skimming later.** Months later you reread only the bold + highlight layers in seconds, then dive into the full text only for the one section you now need.
